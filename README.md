# MyPay of SkyQian

一个拥有高度自适应设备尺寸的网页，用于展示您所有钱包的收款码

<img src="./assets/Payment-Code.png" alt="主界面"  />


## 使用

1. 将此项目克隆到你的项目中

2. 修改`./js/main.js`文件中的第`11、21、31`行代码 将各个`text`参数都换成你自己的钱包二维码内容

3. 修改`./js/footer.js`文件中的第`26`行代码 将`line2.textContent`参数换成你自己的虚拟钱包地址
